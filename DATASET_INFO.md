# Dataset Information

## Dataset Overview
The datasets used in this dissertation were collected from publicly available obesity and health-related data sources for analytical and predictive modelling purposes.

This project focused on identifying obesity-related trends, behavioural patterns, and predictive insights using analytical and AI-supported methods.

---

## Dataset Sources

### 1. Obesity & Health Dataset
Source: Public health and obesity-related datasets from online repositories and research sources.

### 2. Lifestyle & Behavioural Data
Included variables related to:
- Physical activity
- Dietary habits
- BMI indicators
- Lifestyle patterns
- Health-related metrics
